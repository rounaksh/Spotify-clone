## Installation

#### Setup backend .env file

```bash
PORT=...
MONGODB_URI=...
ADMIN_EMAIL=...
CLOUDINARY_API_KEY=...
CLOUDINARY_API_SECRET=...
CLOUDINARY_CLOUD_NAME=...
CLERK_PUBLISHABLE_KEY=...
CLERK_SECRET_KEY=...
NODE_ENV=...
```

#### Setup frontend .env file

```bash
VITE_CLERK_PUBLISHABLE_KEY=...
```

#### Build the app
```bash
npm run build
```
    
#### Run the app
```bash
npm start
```
